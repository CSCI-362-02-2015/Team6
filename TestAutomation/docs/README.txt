This is a testing suite for various python libraries within
the jython open source project

To set up and compile Jython:
    ensure that Mercurial, the Java JDK, and ant are all installed
    In terminal, input the command: hg clone http://hg.python.org/jython
    navigate to the top level directory of Jython in the terminal
    input the command: ant

To run all the tests, from the TestAutomation directory, run:
    python ./scripts/runAllTests.py
